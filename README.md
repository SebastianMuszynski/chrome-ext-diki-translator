# Diki.pl Translator for Chrome

[Diki.pl](https://www.diki.pl) is a great translator, however, it does not have at the moment any browser extension for a quick word lookup. This extension allows to translate any selected word or phrase in a comfortable wayâ€”no more switching tabs!

## How-to

1. Select any word or phrase on the current website.

2. Press `d` twice to open a pop up with the translation.

3. **Hover** over the pop-up area and **leave** in order to close it.

## Languages

It supports **Polish-English** and **English-Polish** translations.
